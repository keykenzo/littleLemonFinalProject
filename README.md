# Little Lemon Ordering App 🍋

The **Little Lemon** app is a functional food ordering application that allows users to browse a Mediterranean menu, filter by categories, and manage their profile information.
<p>This is the final capstone project for the Coursera Meta iOS Developer Professional Certificate. </p>

## Features

* **Onboarding Screen:** A Welcome flow that prompts users to enter their personal details upon their first launch.
* **Stack Navigation:** Implementation of a navigation tabs that allows users to navigate through the app.
* **Profile Screen:** A centralized location to view and manage user account details.
* **Data Persistence:** Ability to save changes in the Profile screen that are retained even after the app is restarted.

## 🛠️ Tech Stack

* **Language:** Swift
* **Framework:** SwiftUI
* **Data Storage:** UserDefaults / Core Data
* **Architecture:** MVVM (Model-View-ViewModel)
